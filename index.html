Items like paragraphs. ```html <p>Paragraph Text</p> ``` Lists ```html <ul> <li>List Item</li> <li>List Item</li> <li>List Item</li> </ul> ``` and images ```html <img src="http://www.fillmurray.com/300/300" alt="Placeholder Image" /> ```

### What Is CSS? CSS (Cascading Style Sheets) are the **style** of your document. They help control the look and feel of the document. You create **selectors** that target **HTML Elements** and then use **CSS Rules** to change the look and feel of your document. Selectors like: p { /* CSS Rules Here */ } Which targets all paragraphs, or: .rotator-slide { /* CSS Rules Here */ } Which targets *any* element with class="rotator-slide", or: #rotator { /* CSS Rules Here */ } Which targets *any* element with id="rotator".

CSS Rules go inside the {curly brackets} that follow the **CSS Selector** and they look like this: 

 p {
	font-size: 2em;
	color: red;
}

Which will apply the font size of 2em and color of red to all paragraphs.

This is a paragraph with the styles above applied to it.

### What Is JS? JS (JavaScript) controls the **behavior** of your document. It allows you to do things such as animate items, hide or show items, react to user input and even process data and perform other general programming tasks. JS can look like this: var myVariable = "Hello World"; That line will create a variable called **myVariable** and assign it a value of **Hello World** (without the quotes). The structure and syntax of the language of JavaScript is much more complex than either HTML or CSS. This is because neither HTML nor CSS are full fledged programming languages. Neither of them have **variables**, **functions**, **objects** or **control structures** such as **if** statements or **loops**. JavaScript has *all* of these.

## Variables ### What *is* a variable? A variable is a named memory location which contains some known or unknown quantity or information. #### In *English* professor. ## Variable Types Variable **values** can be one of several different **types**. The most basic is a **number** var meaningOfLife = 42; // Whole numbers are called 'integers' Javascript can do math with numbers. ```javascript var endingValue = 5 * 2; // the asterisk '*' is the symbol for multiplication. // This will set the value of 'endingValue' to 10. ``` The calculations work the same whether the values are either numbers, or values stored in a variable: ```javascript var startingValue = 5; var multiplier = 2; var endingValue = startingValue * multiplier; // 'endingValue' is still 10. ```
## Variable Types When you store any amount of text, like words or sentances for example, the variable type is a **string** value. ```javascript var myVariable = "Hello World"; ``` You can add strings together to make longer strings ```javascript var first = "Hello"; var second = "World"; var wholePhrase = first + " " + second; ``` The javascript above adds **three** items together: 1. the variable **first** with the value "Hello" * the **anonymous** string value " " * the variable **second** with the value "World" "Hello" + " " + "World" = "Hello World"

## Variable Types You can use the double quote mark (") inside a string that is **delimited** by double quotes by putting a backslash (\\) in front of it. ```javascript var totallyWorks = "Then I said: \"No way you can do that!!!\"."; /* ' /* ^ ^ | | |___________________________________________| | Javascript sees a string starting and ending with the same symbol. ## JavaScript Functions if a variable is a **name** for a **value**, then a function is a **name** for a set of commands or instructions. For example: ```javascript function sayHello(){ /* There can be many commands here in between the two {curly brackets} Let's make the first command an alert to the user. */ alert("Hello World"); } ``` This creates a new **function** called "sayHello" which, when invoked, will execute the **alert** function that comes built into javascipt. The **alert** function is a **Built In Function** and the **sayHello** function is a **User Defined Function**. The only difference between them is that you don't have to write a built in function, it comes for free as a part of JavaScript.
## JavaScript Functions Functions can be used to make your life a LOT easier once you get used to how to write them. ## Using jQuery and the jQuery(); function The basic idea of jQuery is that you target some item on the page and then add or change the behavior of that item. ## How to select an item with jQuery(); ### (So you can alter its behavior) the ```jQuery();``` function takes a single parameter: A string containing a CSS style selector indicating what item, or items, on the page to select. Some examples: ```javascript jQuery("ul li.even"); ``` The line above selects all list items, with a class of "even" in an any unordered list. ```javascript jQuery("#rotator .slide:first"); ``` The line above selects the first item with a class of "slide" inside the item with an id of "rotator".
## How to select an item with jQuery(); ### (So you can alter its behavior) The HTML items or your page that match your selector are added into something called the **wrapped set**. This is a special grouping of items that jQuery uses to know what items to alter or add behavior to. When you run a jQuery **method** the items in the **wrapped set** are the ones that will be altered or effected.

## Hover and Click Events Setting the width of something is pretty cool but we could use CSS to do that. Let's try something only JavaScript could help us with. Let's add a **click event**. When we add a function to a **user triggered event** like this we call it an **event listener**. Our code just sits around waiting and listening for the event to happen. When someone finally clicks the item it triggers our function to run.

## So, What IS an image rotator? At it's most basic, a rotator is a list of images that uses javascript to show one image on its own, then another, then another... SO if we used HTML like this: ```html <div id="rotator"> <ul class="slides"> <li><img src="img/rotator/image1.jpg" alt="First Slide"></li> <li><img src="img/rotator/image2.jpg" alt="Second Slide"></li> <li><img src="img/rotator/image3.jpg" alt="Third Slide"></li> </ul> </div> ``` The javascript would show the first and hide the rest. Then, after a couple of seconds, show the second and hide the rest. Then keep going until all of the slides have been shown. Then start again at the beginning.
